# 📌 پینترست با اینترنت ملی 
با استفاده از این پروژه شما میتوانید با نت ملی ایران درون پینترست جستجو کنید و هر محتوایی از پینترست را دانلود کنید. 

# 🧪 آموزش استفاده 
- وارد اکانت گیتهاب خود شوید.
- به این صفحه مراجعه کنید و از بالای صفحه گزینه ی fork را بزنید.
- یک نام برای fork خود انتخاب کنید و سپس create fork را بزنید.
- پروژه شما آماده است به فورک ساخته شده در گیتهاب خود مراجعه کنید.
- به سربرگ Actions بروید و سمت راست زیر all workflows از امکانات موجود در این ربات استفاده کنید.
